## 🔧 **Проблемы и решения:**

### Проблема: "VPN не подключается"
**Проверить:**
- Конфиг правильного формата?
- VPN permission выдано?
- libxivpn.so на месте?

### Проблема: "Gradle build failed"
**Решение:**
- Build → Clean Project
- Build → Rebuild Project

### Проблема: "Приложение крашится при запуске"
**Проверить Logcat:**
```bash
adb logcat | grep -E "FATAL|Exception"
```
