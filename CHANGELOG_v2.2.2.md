# Changelog v2.2.2 - Traffic Format Fix

## Дата: 20 декабря 2025

## Основные изменения

### Исправлено форматирование трафика во всех компонентах UI

#### 1. Обновлен `SubscriptionInfo.kt`:
- `formatBytesCompact()`: Конвертация больших MB в GB (822MB → 0,8GB)
- `formatBytes()`: Применена та же компактная логика к обычному форматированию
- `formatTrafficForNotification()`: Использует компактные версии для уведомлений
- Убраны ненужные десятичные знаки для целых чисел (100,00GB → 100GB)

#### 2. Обновлен `KizVpnService.java`:
- `formatBytes()`: Реализовано компактное форматирование в VPN сервисе
- Автоматическая конвертация MB >= 100 в формат GB
- Убраны десятичные знаки для целых чисел (1000,0GB → 1000GB)

#### 3. Улучшения отображения трафика:
- **Модальное окно**: `822,55 MB / 100,00 GB` → `0,8GB/100GB`
- **Меню подписки**: `0,8GB/100,0GB` → `0,8GB/100GB`
- **Шторка уведомлений**: `822,93 MB / 100,00 GB` → `0,8GB/100GB`
- Единообразный компактный формат во всех компонентах UI

#### 4. Сохранено:
- Форматирование русской локали (запятая как разделитель десятичных)
- Логика отображения безлимитных подписок
- Все изменения протестированы и успешно скомпилированы
