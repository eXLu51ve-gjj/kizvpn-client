# 🎯 KIZ VPN - Полный список функций

## ✨ Основные возможности

### 1. **Поддержка протоколов**
- ✅ **VLESS** (Xray-core)
  - Reality (с публичным ключом)
  - WebSocket
  - TLS/XTLS
  - TCP/gRPC транспорты
  - Flow контроль (xtls-rprx-vision)
  
- ✅ **WireGuard**
  - Современный VPN протокол
  - Высокая скорость
  - Низкая задержка
  - PreSharedKey поддержка

---

### 2. **Интерфейс и UX**

#### Главный экран:
- 🎨 Современный темный дизайн
- 🎥 Видео фон с анимацией

- 📊 Статистика трафика (загрузка/выгрузка) в реальном времени
- ⏱️ Время подключения
- 📱 Адаптивный UI для разных экранов

#### Навигация:

- 📜 История подключений (History)

### 6. **VPN функциональность**

#### Подключение:
- 🚀 Быстрое подключение (1-2 секунды)
- 🔒 Создание TUN интерфейса
- 🌐 Маршрутизация всего трафика через VPN
- 🛡️ DNS leak protection (8.8.8.8, 8.8.4.4)

#### Управление:
- ▶️ Подключение/отключение одной кнопкой
- 🔄 Автоподключение при запуске
- 💾 Сохранение последнего конфига
- 🔌 Автоматическое переподключение при обрыве

#### Безопасность:
- 🔐 Защита сокетов от утечек
- 🛡️ Foreground service для стабильности
- 🚫 Блокировка трафика при разрыве
- ✅ VPN permission запрос

## 🏗️ Технические характеристики

### Архитектура:
- 📱 Android 8.0+ (API 26+)
- 🏗️ MVVM архитектура
- 🎨 Jetpack Compose UI
- 🔄 Kotlin Coroutines
- 💾 SharedPreferences хранилище

### VPN Ядро:
- ⚙️ libxivpn.so (форк Xray-core)
- 🔌 IPC коммуникация (LocalSocket)
- 🛡️ TUN интерфейс
- 🌐 Full routing (0.0.0.0/0)

### Поддерживаемые архитектуры:
- ✅ arm64-v8a (64-bit ARM)
- ✅ x86_64 (64-bit Intel/AMD)
- ⚠️ armeabi-v7a (32-bit ARM) - частично

### Библиотеки:
- Jetpack Compose
- Material3
- OkHttp (HTTP клиент)
- Gson (JSON парсер)
- Accompanist (permissions, animations)
- ZXing (QR код сканер)
- Lottie (анимации)
